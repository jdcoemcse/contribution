# JDCOEM CSE Contribution Repository

This repository exists to make contributing study materials easier for CSE students.

## Why this Repository?

The main syllabus repository contains study materials, notes, question papers, and resources for the CSE Batch 2023–2027:

https://github.com/jdcoemcse/syllabus

As the repository has grown over the years, its size has become quite large (around 3 GB). For many students, especially beginners, forking and cloning such a large repository just to contribute a few notes is inconvenient.

To solve this problem, this lightweight contribution repository was created.

## Purpose

The goal of this repository is to:

- Collect notes, question papers, practicals, and other academic resources from different CSE batches.
- Allow students to learn Git and GitHub through real contributions.
- Reduce the need to fork and clone the large syllabus repository.
- Build a centralized collection of resources that can later be added to the main syllabus repository.
- Help future students understand the curriculum structure and access useful study materials more easily.

## Repository Structure

```
Batch 2024-2028/
Batch 2025-2029/
```

Each batch can contribute materials related to its own syllabus and semesters.

## How Contributions Work

1. Fork this repository.
2. Clone your fork to your computer.
3. Add your notes, question papers, practical files, or other useful academic resources in the appropriate batch folder.
4. Commit your changes with a meaningful commit message.
5. Push the changes to your fork.
6. Create a Pull Request.

By contributing here, students can practice:

- Forking repositories
- Cloning repositories
- Creating commits
- Pushing changes
- Opening Pull Requests
- Collaborating using GitHub

## Relation with the Main Repository

This repository is not a replacement for the main syllabus repository.

The main repository remains the central collection of study materials:

https://github.com/jdcoemcse/syllabus

This repository acts as a contribution hub where students can easily submit new resources. Approved materials can later be organized and added to the main repository for long-term preservation.

## Who Can Contribute?

Any CSE student who wants to help future batches by sharing:

- Class Notes
- Question Papers
- Practical Files
- Lab Manuals
- Assignments
- Reference Materials
- Important Resources

## A Small Request

Many students struggle to organize their academic resources. This repository is an attempt to make useful materials available in one place so that future students can spend less time searching and more time learning.

Every contribution, no matter how small, can help someone in the future.

Happy Contributing! 🚀
